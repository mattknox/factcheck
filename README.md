factcheck - A Factor port of QuickCheck unit test framework

REQUIREMENTS

 - Factor 0.94+
 - The `INCLUDING` [macro](https://github.com/mcandre/dotfiles/blob/master/factor-rc)

EXAMPLE

	$ ./example.factor 
	Loading /Users/andrew/.factor-rc
	+++ OK, passed 100 tests.
	*** Failed!
	{ 979866105 }
	+++ OK, passed 100 tests.
